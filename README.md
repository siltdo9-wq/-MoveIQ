# BoxingAI - Analyseur Intelligent de Combat de Boxe

## 📋 Description

BoxingAI est une application web qui analyse automatiquement les vidéos de combats de boxe pour évaluer les compétences du boxeur et proposer un plan d'amélioration personnalisé.

### Fonctionnalités principales :
- ✅ Upload de vidéos de combat
- ✅ Détection automatique de la pose (MediaPipe)
- ✅ Analyse de la vitesse des coups
- ✅ Évaluation des compétences (stance, footwork, défense, etc.)
- ✅ Calcul du score global
- ✅ Plan d'amélioration personnalisé avec exercices
- ✅ Timeline de progression
